Você é um Arquiteto de Software Sênior especialista em Python, FastAPI, Docker, APIs REST e Inteligência Artificial Generativa.

Desenvolva uma aplicação completa chamada **Local LLM Server**.

## Objetivo

Criar um servidor de Inteligência Artificial executado 100% localmente, capaz de disponibilizar modelos de linguagem (LLMs) através de uma API REST compatível com a OpenAI.

O objetivo é permitir que qualquer aplicação desenvolvida por mim (.NET, Python, Node.js, React, etc.) possa consumir uma IA local exatamente como consumiria a API da OpenAI, bastando alterar a Base URL da conexão.

Toda a solução deve funcionar completamente offline após a instalação dos modelos.

---

## Requisitos

A aplicação deverá:

- Executar modelos locais utilizando Ollama.
- Disponibilizar uma API REST.
- Ser compatível, sempre que possível, com a API da OpenAI.
- Permitir que qualquer aplicação externa utilize a IA sem conhecer o Ollama.
- Possuir arquitetura desacoplada para permitir substituir futuramente o Ollama por outro provedor (LM Studio, vLLM, llama.cpp, etc.).
- Funcionar integralmente em Docker.

## Compatibilidade

A API deve seguir o padrão da OpenAI sempre que possível.

Um projeto que atualmente utiliza:

https://api.openai.com/v1

deverá conseguir utilizar:

http://localhost:8000/v1

alterando apenas a Base URL e, quando necessário, a API Key.

As estruturas de Request e Response devem ser compatíveis com a OpenAI.
